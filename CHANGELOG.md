# Changelog

All notable changes to this project will be documented in this file.

## [v0.7.0] - 2026-06-23 · 공유 배포 `@13`

### ✨ Features
- 끝내기/자동 종료 되돌리기(종료 취소) — 종료 시 잠긴 패널 밖(헤더)에 [↩ 종료 취소] 버튼 추가. 히스토리 스냅샷에 종료 상태(gameOver/endReason)를 포함해 Undo가 종료까지 복원하고, 자동 종료(끝내기/정규)는 유발 플레이를 함께 되돌리며 prevHalfRef를 복원 하프이닝으로 맞춰 즉시 재종료 방지. 수동 종료는 잠금만 해제 (1f9400c)
- 수동 득점 -1 (자동 득점 정정) — 3루 주자가 안타에 자동 득점 처리됐으나 실제로는 홈에서 막힌 경우 등을 정정. 현재 공격 팀의 현재 이닝 점수를 차감(0 이하 방지), 주자 복귀는 [3루 주자] 토글 사용 (7068fc2)

### 📝 Documentation
- 공유 배포 버전 표기 `@11` → `@13` 갱신 (README·CHANGELOG)

## [v0.6.0] - 2026-06-22 · 공유 배포 `@11`

### ✨ Features
- 경기 자동 종료(끝내기/정규) — 최종 이닝(정규 이닝 수) 이상 말(홈) 공격에서 홈이 앞서면 즉시 종료(끝내기, 말 시작 시 이미 리드면 말공격 생략). 말 공격 종료 후 동점이 아니면 정규 종료, 동점이면 연장. 점수·이닝이 한 커밋으로 반영된 뒤 평가되도록 useEffect로 판정하고 하프이닝 경계에서만 정규 종료를 적용해 초공격 중 리드 오작동 방지. 종료 사유로 끝내기/승팀 배너 표시 (74c33e9)
- 진행 중 경기 자동저장·이어하기 — 모든 동작마다 전체 경기 상태를 localStorage에 저장, 재접속/새로고침 시 '이어서 진행/새 경기' 프롬프트로 복원. 무진행 상태는 프롬프트 생략, localStorage 차단은 무시(기능만 비활성) (74c33e9)
- 최종회/연장 배지 — 헤더 이닝 표시에 `최종회`(정규 이닝 도달)·`연장`(초과) 배지를 노출해 설정 이닝 인지 상태를 가시화 (c8d05ed)

### 📝 Documentation
- 공유 배포 버전 표기 `@9` → `@11` 갱신 (README·CHANGELOG)

## [v0.5.0] - 2026-06-22 · 공유 배포 `@9`

### ✨ Features
- 팀별 참가 인원 분리 — 단일 `rosterSize`를 `awayRosterSize`/`homeRosterSize`로 나눠 어웨이·홈 인원을 서로 다르게 운영. 명단 편집기를 좌(어웨이)·우(홈) 독립 2열로 재구성: 팀별 참가 인원 표시, 행별 ✕ 삭제(해당 팀 타순만 당겨짐), ＋ 타자 추가. 타순 순환·리더보드·라인업 카드·CSV/웹훅 내보내기 모두 팀별 인원으로 동작. 경기 규칙의 단일 '타자 수' 입력은 팀별 편집기로 일원화 (14cc36c)

### 🐛 Bug Fixes
- 결과 전송 URL 검증 강화 — 미치환 템플릿(`<?= webAppUrl ?>`)·비정상 주소를 차단해 정적(미리보기) 호스팅에서의 거짓 "전송 완료" 방지. no-cors 특성상 저장 성공을 직접 확인할 수 없으므로 "전송 요청됨 + 시트에서 확인" 문구로 정직하게 안내. (저장이 안 되던 주원인은 로그인 필수인 `@HEAD` URL 사용 — 공유 `/exec`에서만 익명 저장 동작) (14cc36c)

### 💄 Style
- 헤더 버튼 정렬 정리 — 제목+이닝 줄과 버튼 줄을 세로 분리, 버튼을 전체 폭 그리드(모바일 2×2·sm+ 1×4)로 균등 배치해 좁은 중앙 컬럼 넘침/잘림 해소 (0bf4723)
- 액션 패널 버튼 정렬 정리 — 모든 버튼 동일 `min-h` + 중앙 정렬로 1~2줄 라벨 높이 통일, `h-full`·`justify-end`·`mt-auto` 제거로 4개 컬럼 상단 정렬 (261a6df)

### 📝 Documentation
- 공유 배포 버전 표기 `@7` → `@9` 갱신 (README·CHANGELOG)

## [v0.4.0] - 2026-06-22 · 공유 배포 `@7`

### ✨ Features
- 병살(더블플레이) 기록 패널 추가 — 뜬공 포구 후 주자 베이스 미리 이탈 등 더블플레이 상황 기록. 주자가 있을 때만 활성화되며 베이스에 있는 주자를 선택하면 타자 뜬공/땅볼 아웃(타수+1, 안타 없음) + 지정 주자 아웃 = 2아웃 처리, Undo 지원 (ed97fd9)
- 시트 저장 없이 경기 종료 옵션 추가 — 헤더에 [🚫 저장 없이 종료] 버튼 신설(친선·연습 경기 등 누적 불필요 시). 확인 후 패널만 잠그고 구글 시트 전송은 하지 않으며, 종료 후에도 [📤 결과 저장]으로 수동 저장 가능. 기존 [경기 종료]는 [🏁 종료·저장]으로 라벨 정리

### 📝 Documentation
- 테스트/툴링 부재 및 참고 문서 안내로 CLAUDE.md 보강 — 빌드 외 npm 스크립트·테스트·린터 부재 명시(전역 커버리지 컨벤션 비적용), 빌드 중간산출물/커밋 산출물 구분, 한국어 참고 문서 목록 추가 (6b9778d)
- 기록용 링크 모음 문서 추가 (b3f39d3)
- 공유 배포 버전 표기 `@6` → `@7` 갱신 — 병살 패널·시트 저장 없이 종료 옵션 재배포 반영 (9bf9b0a)
