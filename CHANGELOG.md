## [v0.1.0] - 2026-06-18

### ✨ Features
- React 단일 HTML 스코어보드(Index.html)를 Apps Script `doGet`으로 서빙 (902ea92)
- `doPost`로 경기 결과를 구글 시트(경기요약/타자기록)에 누적 저장 (902ea92)
- 결과 전송 기본 URL을 배포된 웹앱 `/exec` 주소로 자동 주입 (902ea92)

### 🐛 Bug Fixes
- 3아웃 시 타순이 다음 타자로 넘어가지 않던 버그 수정 (902ea92)
